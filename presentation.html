Smart Try-On Pipeline

Intelligent outfit conflict resolution ensures natural, realistic results every time

Pipeline Steps

1. Product Analysis Nova Lite
Classifies garment type: upper body, lower body, full body, footwear, cosmetics
2. Outfit Classification Nova Lite
Analyzes what the user is currently wearing to detect conflicts
3. Smart Prompt Generation
Builds context-aware instructions based on garment + outfit combination
4. Virtual Try-On Gemini 2.5 Flash
Generates photorealistic result preserving user's identity

Conflict Resolution Examples

User wearing dress + trying on a top

AI removes dress, applies the top, and generates matching pants/skirt that complement the new top

User wearing dress + trying on pants

AI removes dress, applies the pants, and generates a matching top that complements the bottoms

Color variation switch

Auto-detects swatch changes and re-runs try-on instantly with the new color

Identity preservation

Preserves user's exact face, skin tone, body shape, and hairstyle across all try-ons
